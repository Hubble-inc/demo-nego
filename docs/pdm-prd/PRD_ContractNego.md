## プロダクト概要

NegoSignは、契約書の交渉から署名・締結までを一つの画面で完結する契約交渉プラットフォーム。AIが契約書を平易な言葉で解説し、リアルタイムチャットで交渉を進め、電子署名で締結する。

| 項目 | 説明 |
|------|------|
| 対象ユーザー | 契約書を扱う全てのビジネスパーソン |
| 契約書の管理 | NegoSign内で完結 |
| 入口画面 | NegoSignダッシュボード |
| バージョン管理 | NegoSign内で完結 |
